# SHOPNOW

A modern e-commerce web application where users can browse products, manage a shopping cart, and complete checkout.

## Features

- Browse and search products by title or tag
- Product detail pages with reviews and ratings
- Shopping cart with localStorage persistence
- Checkout flow with order summary and VAT calculation
- Responsive design with mobile navigation

## Prerequisites

- Node.js (v20+)
- npm

## Getting Started

### Installation

```bash
npm install
```

### Running the project

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
